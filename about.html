Jonas is a PhD-candidate in biophysics at the Norwegian University of Science and Technology. He does research in the field of microfluidics - making microchips which can automate tasks in biomedicine for point-of-care diagnostics.

He is also the developer of Highlights, a Mac app for extracting PDF annotations to MultiMarkdown.

This is a Jekyll website with static content written in Markdown. The website is hosted on GitHub pages. The design is done using the Twitter Bootstrap framework. The icons are from Font Awesome by Dave Gandy.
